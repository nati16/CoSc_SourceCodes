
                             Railway Management System
We developed this program for the railway management.

What it basically does is that it has two accounts, Admin and customer

In the Admin mode, we designed this for the administrator to do

1.Add train
2.Employee Management

In the employee management

We added the following functionalities

	1. Build the employee table
	2. List the employee table
	3. Search the employee table
	4. Backup the the employee table
	5. Restore [Backup file] the employee table

In the customer mode, we designed this for the customer to do

1.Reserve Seats
2.Show available trains
